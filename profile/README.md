# Hi, I'm Matt 👋

Building web apps on Cloudflare Workers, iOS/macOS apps, and a handful of
Python CLIs and Claude skills.

### What's here

- **Web** — React + TypeScript + Vite on Cloudflare Workers
- **Mobile / desktop** — native iOS & macOS (SwiftUI)
- **Tooling** — Python CLIs, several packaged as Claude skills

Shared CI lives in this repo: reusable GitHub Actions workflows
(`reusable-*.yml`), a default `SECURITY.md`, and the canonical Renovate
config that the other repos extend.
